Tornado is a tiny open source IMU built for FRC! The design includes many features such as...
* __CAN FD support__
* __Efficient buck converter__
* __RGB LEDs__
* __EEPROM for persistent config__
* __Reverse voltage protection__
* __23mm x 23mm size__
